Projects

A selection of my work

Last year Mountain Rescue England and Wales had to respond to over 3000 callouts in the countryside... Ramblr is an app designed to help curb those figures. With Ramblr you are able to log in, set a desired time for your ramble and add a trusted contact's phone number. If you don't alert the app that you have completed your walk, Ramblr will automatically send a text message at the end of your set time, to your trusted contact, with a live location of your whereabouts.

Tech used

JavaScript | AWS Amplify | AWS Cognito | AWS DynamoDB | AWS Lambda | AWS CloudWatch | Axios | React Native | Expo | CSS | SMSWorks API

This was the final project I worked on as part of a team where were able to put everything we had learned during the Bootcamp with Northcoders in action and explore new technologies.
